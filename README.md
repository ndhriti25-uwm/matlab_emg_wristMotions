# matlab_emg_wristMotions
# EMG Wrist Motion Analysis (MATLAB)

This project analyzes surface EMG (sEMG) signals collected during wrist motions.
The goal is to extract features (e.g., RMS) from EMG recordings and compare muscle
activation across different gestures and trials.
